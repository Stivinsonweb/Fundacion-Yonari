# Fundación Yonari

## Descripción
Sitio web de la Fundación Yonari, una organización dedicada a brindar oportunidades a quienes más lo necesitan. Nuestra misión es crear un impacto positivo y duradero en las comunidades, construyendo puentes hacia un futuro mejor a través de la educación, el apoyo social y el desarrollo integral de las personas.

## Tecnologías Utilizadas
- **PHP**: Backend y estructura del sitio
- **Tailwind CSS**: Framework CSS para el diseño
- **HTML5**: Estructura y contenido
- **JavaScript**: Interactividad

## Estructura del Proyecto
```
Fundacion-Yonari/
├── Src/
│   ├── assets/       # Recursos (imágenes, videos, etc.)
│   ├── inc/          # Archivos incluidos (header, footer, head)
│   └── Pages/        # Páginas del sitio
├── Index.php         # Página principal
└── README.md         # Este archivo
```

## Requisitos
- XAMPP (o cualquier servidor con PHP)
- PHP 7.4 o superior
- Navegador web moderno

## Instalación

1. Clona o copia el proyecto en la carpeta `htdocs` de XAMPP:
   ```bash
   C:\xampp\htdocs\Fundacion-Yonari
   ```

2. Inicia Apache desde el panel de control de XAMPP

3. Abre tu navegador y accede a:
   ```
   http://localhost/Fundacion-Yonari
   ```

## Características
